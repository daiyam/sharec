This configuration package includes:

* `eslint` with `standard` preset and `prettier` plugin
* `prettier`
* `lint-staged`
* `husky`
* `commmit-lint`
* `commitizen`

Install it by:

```shell
npm i --save-dev sharec-essential-config
```

Or with `yarn`:

```shell
yarn add -D sharec-essential-config
```
